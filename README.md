游戏操作说明

本文档介绍本游戏支持的键盘与手柄操作方式。游戏内包含两种主要情境：角色控制（横版移动/跳跃）与关卡选择界面。以下操作均基于默认按键映射。

---

键盘操作

游戏内（角色控制）

功能 按键
向左移动 A 或 ←（左方向键）
向右移动 D 或 →（右方向键）
跳跃 空格键（Jump）
暂停游戏 Esc

关卡选择界面

功能 按键
移动光标（选择关卡） WASD 或 方向键（上/下/左/右）
确认选择 / 载入场景 空格键（Jump） （移动至该关卡/载入场景）

---

手柄操作

（以 Xbox 手柄为例，兼容同类布局；其他手柄请参照对应键位）

游戏内（角色控制）

功能 手柄操作
向左/右移动 左摇杆 左右方向
跳跃 A 键（跳跃键 / Jump）
暂停游戏 Y 键

关卡选择界面

功能 手柄操作
移动光标（选择关卡） 左摇杆 上/下/左/右
确认选择 / 载入场景 A 键（跳跃键 / Jump） （移动至该关卡/载入场景）

---

补充说明

· 移动轴：游戏内角色仅支持水平方向移动（左右），因此左摇杆或 A/D 键仅响应左右输入。
· 暂停功能：在任何界面（包括关卡选择）按下 Esc（键盘）或 Y（手柄）均可暂停游戏。
· 手柄兼容性：若使用非 Xbox 布局手柄，请根据按键位置对应功能：
  · 跳跃键通常为下方主动作键（如 PlayStation 的 ✕ 键）。
  · 暂停键通常为上方右侧功能键（如 PlayStation 的 △ 键）。
